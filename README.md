# JeshCuhh Quiet Luxury Site

This package is organized as a responsive personal archive website for my web design class.

## Structure

- `index.html`: homepage
- `style.css`: shared quiet-luxury / old-money visual system
- `script.js`: navigation, accordion, and reveal interactions
- `beauty-self-care/`: beauty routines and protocols
- `wardrobe/`: capsule wardrobe, clothing care, shoe care, jewelry
- `home-lifestyle/`: decor, masstige, procedure hierarchy
- `finance-investing/`: FIRE/churning pointers, investing routine, interactive spending graph
- `portfolio/`: modeling portfolio
- `assets/images/`: cleaned web-safe image filenames

## Notes

- Folder names and URLs use lowercase kebab-case.
- The homepage keeps the protocol accordion plus button interaction.
- The expanded protocol panels contain short previews and Learn More buttons.
- Full details live on dedicated pages to keep the homepage clean and readable.
- Design direction: quiet luxury, old money, warm neutrals, champagne borders, serif headings, restrained motion.
